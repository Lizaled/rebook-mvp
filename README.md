Описание проекта

**ReBook** — это веб-платформа для безопасного обмена книгами внутри локальных сообществ (школ, семей, предприятий). Проект направлен на поддержку культуры чтения, устойчивого потребления (ESG) и снижение углеродного следа за счет повторного использования печатной продукции.

Проект реализован как MVP (Minimum Viable Product) в рамках курса "Системы искусственного интеллекта" СФУ.

Ключевые функции MVP:
1. **Публичный каталог:** Просмотр доступных книг без регистрации.
2. **Аутентификация:** Регистрация и вход пользователей.
3. **Личный кабинет:** Профиль пользователя со статистикой эко-вклада (сэкономленный CO₂).
4. **Добавление книг:** Форма публикации книги с выбором точки выдачи.
5. **Админ-панель:** Управление данными через встроенный интерфейс Django Admin.

---

Технологический стек

*   **Backend:** Python 3.10+, Django 5.x
*   **Database:** SQLite (для разработки), готова к миграции на PostgreSQL
*   **Frontend:** HTML5, CSS3, Bootstrap 5 (адаптивный дизайн)
*   **Tools:** Git, GitHub

Установка и запуск

Предварительные требования
*   Установленный Python 3.10 или выше
*   Pip (менеджер пакетов Python)

Инструкция по локальному запуску

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШ_НИКНЕЙМ/rebook-mvp.git
    cd rebook-mvp
    ```

2.  **Создайте виртуальное окружение и активируйте его:**
    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Выполните миграции базы данных:**
    ```bash
    python manage.py makemigrations
    python manage.py migrate
    ```

5.  **Создайте суперпользователя (администратора):**
    ```bash
    python manage.py createsuperuser
    # Следуйте инструкциям в терминале (введите username, email, password)
    ```

6.  **Запустите сервер разработки:**
    ```bash
    python manage.py runserver
    ```

7.  **Откройте проект в браузере:**
    *   Главная страница (Каталог): [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
    *   Админ-панель: [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
